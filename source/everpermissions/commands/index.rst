La liste des commandes du plugin EverPermissions.

Générale
~~~~~~~~

Gestion du plugin
-----------------
- **Commande :** ``/everpermissions <help|reload|migrate [sql|conf]>``
- **Description :** 
	- **help :** Affiche l'aide du plugin
	- **reload :** Recharge le plugin
	- **migrate :** Transfère la base de données
- **Permissions :** 
	- ``everpermissions.commands.execute``
	- ``everpermissions.commands.help``
	- ``everpermissions.commands.reload``
	- ``everpermissions.commands.migrate``
